# AI-Powered Legal Contract Analyzer

A Streamlit app that analyzes PDF contracts using RAG and Groq's Llama 3.1 70B.

## Features
- Upload PDF contracts
- Executive summary, key clauses, entity extraction
- Privacy-focused (user provides own API key)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `streamlit run contr.py`

## Deployment
Ready for Streamlit Community Cloud or Hugging Face Spaces.
